Decompile

发表于 2019-04-08

java -g Test4.java

默认情况下局部变量表属于调试级别的信息，javac 编译的时候并没有编译进字节码，

我们可以加上 javac -g 生成字节码的时候同时生成所有的调试信息。

javac -c Test4.class

对类进行反编译。

javac -s Test4.class

输出签名的类型描述符。

javac -v Test4.class

输出更多详细的信息，比如栈大小、方法参数的个数。

javac -p Test4.class

默认情况下，javap 会显示访问权限为 public、protected 和默认（包级 protected）级别的方法，

加上 -p 选项以后可以显示 private 方法和字段。

javac -l Test4.class

打印字节码行号和局部变量表变量表。
